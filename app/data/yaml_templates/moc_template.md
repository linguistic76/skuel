# Yoga Fundamentals

A Map of Content (MOC) is a KU that organizes other KUs. It creates navigational
structure without imposing rigid hierarchy — the same KU can appear in multiple MOCs.

## How This Map Works

This MOC organizes four foundational yoga knowledge units in a suggested learning
order, but each can be explored independently.

1. **Posture Basics** — physical alignment and body awareness
2. **Breath Control** — pranayama techniques
3. **Meditation Intro** — settling the mind through breath
4. **Yoga Philosophy** — the ethical and philosophical foundation
